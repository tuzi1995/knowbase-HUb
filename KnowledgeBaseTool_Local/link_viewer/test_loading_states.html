🎨 加载状态测试页面

测试 K-Matrix 助手的所有加载状态组件

📦 骨架屏（Skeleton Screen） 推荐用于首次加载
骨架屏提供内容结构预览，减少感知等待时间，适合表格、卡片、列表的首次加载。
LoadingManager.showSkeleton('container', 'table', 5); // 数据加载完成后会自动替换骨架屏
⚡ 顶部进度条（Top Progress Bar） 轻量级
顶部进度条不遮挡内容，适合页面切换、后台同步等不阻塞用户操作的场景。
LoadingManager.showTopProgress(); await fetchData(); LoadingManager.hideTopProgress();
🔄 表格加载遮罩（Table Loading Overlay） 阻塞操作
表格加载遮罩明确提示加载状态，防止用户重复操作，适合刷新、筛选、排序等场景。
ID 产品名称 状态 更新时间
001 产品 A 正常 2026-04-22
002 产品 B 处理中 2026-04-22
LoadingManager.showTableLoading('tableBody', '正在加载数据...'); await loadData(); LoadingManager.hideTableLoading('tableBody');
🔘 按钮加载状态（Button Loading） 防止重复提交
按钮加载状态防止用户重复点击，提供明确的操作反馈，适合表单提交、保存、删除等操作。
LoadingManager.setButtonLoading(button, true); await saveData(); LoadingManager.setButtonLoading(button, false);
📝 内联加载（Inline Loading） 轻量级
内联加载适合小区域的加载提示，如搜索结果、下拉列表等。
LoadingManager.showInlineLoading('container', '搜索中...');
📭 空状态提示（Empty State） 友好提示
空状态提示用于数据为空、搜索无结果、加载失败等场景，提供友好的提示和操作建议。
LoadingManager.showEmptyState('container', { icon: '📭', title: '暂无数据', description: '当前没有可显示的内容', actionText: '刷新', onAction: () => refresh() });
🌐 全屏加载遮罩（Fullscreen Loading） 阻塞全部操作
全屏加载遮罩用于重要的、耗时的操作，如数据同步、批量处理等，完全阻止用户操作。
LoadingManager.showFullscreenLoading('正在同步数据，请稍候...'); await syncData(); LoadingManager.hideFullscreenLoading();
🎁 包装函数（Wrapper Functions） 推荐使用
使用包装函数可以自动处理加载状态的显示和隐藏，代码更简洁。
// 包装异步函数 const loadDataWithLoading = LoadingManager.withSkeleton( 'container', async () => { const data = await fetchData(); renderData(data); }, 'table', 5 ); // 调用时自动显示/隐藏加载状态 await loadDataWithLoading();
