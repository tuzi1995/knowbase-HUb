按钮系统

三种按钮样式，每种都有明确的用途和视觉层级。

Primary Button

Secondary Button
<button class="btn-secondary-outline"> <span>了解更多</span> </button>
Ghost Button
<button class="btn-ghost"> <span>取消</span> </button>

输入框系统

现代化的输入框，带有聚焦光环和悬停效果。

Text Input
<input type="text" class="input-modern" placeholder="请输入文本...">
Select
<select class="input-modern"> <option>选项 1</option> <option>选项 2</option> </select>
Textarea
<textarea class="input-modern" placeholder="请输入多行文本..." rows="3"></textarea>

卡片系统

两种卡片样式：标准卡片和渐变边框卡片。

标准卡片

纯白背景，带有阴影和边框。悬停时会上浮。

渐变边框卡片

带有 Electric Blue 渐变边框的特色卡片。

渐变覆盖卡片

悬停时显示渐变覆盖层的卡片。
